<h2 style="padding-left: 60px"> Usage : </h2>

```go
    package main

    import (
        "euler"
        "fmt"
    )

    func main() {
        fmt.Println(euler.LargestPalindromeProduct(2))
        fmt.Println(euler.LargestPalindromeProduct(3))
    }
```

```console
    $ go run .
    9009
    906609
    $
```
